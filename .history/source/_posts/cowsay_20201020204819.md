### 引入

npm install -g cowsay

### 使用

在命令行敲： cowsay 随便文字字母啥的

- cowsay -l ： 查看其他动物

- cowsay -f 其他动物代号 动物说的话 ：使用其他动物

在代码注释里秀：

```
var cowsay = require("cowsay");

console.log(cowsay.say({
	text : "I'm a moooodule",
	e : "oO",
	T : "U "
}));

// or cowsay.think()

```
